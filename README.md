# Digiga Gacha Card Game

Dự án game gacha thẻ bài sử dụng Google Sheets làm Database và Google Apps Script làm Backend.

## Cấu trúc dự án
- `index.html`: Giao diện game (Gacha UI).
- `digiga.js`: Code xử lý logic (đang cập nhật).

## Hướng dẫn
1. Kết nối Google Sheet với Apps Script.
2. Triển khai Apps Script dưới dạng Web App.
3. Dán URL Web App vào file giao diện.
